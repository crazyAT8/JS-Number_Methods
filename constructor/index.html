JavaScript Number property: constructor

    Definition:
  • the constructor property returns the function that created the Number prototype.
  • For JavaScript numbers the constructor property returns: function Numbers() { [native code] }

Syntax

number.constructor

Return Value
function Number() { [native code] }
    Notes:
  • constructor is an ECMAScript1 (ES1) feature.
  • ES1 (JavaScript 1997) is fully supported in all browsers.

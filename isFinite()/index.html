JavaScript Number Method: Number.isFinite()

    Definition:
  • this method returns true if a number is a finite number.
  • Infinite (not finite) numbers are Infinity, -Infinity, or NaN
  • Otherwise it returns false.

Syntax

Number.isFinite(value)

Similar to:

  • Global isFinite() method
  • Global isNaN() method
  • Number.isNaN() method
  • POSITIVE INFINITY Property
  • NEGATIVE INFINITY Property
Parameters
Parameter Description
value Required.
The value to be tested.
Return Value
Type Description
A boolean true if the value is a finite Number, otherwise false.
    Notes:
  • Difference Between isFinite() and Number.isFinite()
    • isFinite() returns true if a value is a finite number.
    • Number.isFinite() returns true if a number is a finite number.
    • In other words: isFinite() converts the value to a number before testing it.
  • ECMAScript6 (ES6) feature
  • ES6 (JavaScript 2015) is supported in all modern browsers since June 2017
  • is not supported in Internet Explorer

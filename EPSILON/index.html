JavaScript Number Method: EPSILON

    Definition:
  • this method returns the difference between the smallest floating point number greater than 1 and 1.
  • Number.EPSILON has the value of 2.220446049250313e-16

Syntax

Number.EPSILON()

Return Value
Type Description
Number 2.220446049250313e-16
    Notes:
  • EPSILON is a property of the JavaScript Number object.
  • You can only use it as Number.EPSILON
  • Using x.EPSILON, where x is a variable, will return undefined.
  • Number.EPSILON is an ECMAScript6 (ES6) feature.
  • ES6 (JavaScript 2015) is supported in all modern browsers since June 2017
  • Number.EPSILON is not supported in Internet Explorer.

JavaScript Number Method: isNaN()

    Definition:
  • Short for "Not-a-Number"
  • NaN is a number that is not a legal number.
  • The Number.isNaN() method returns true if the value is NaN, and type is a Number.

Syntax

Number.isNaN(value)

Similar to:

  • NaN() Property
  • isNaN() global method
Parameters
Parameter Description
value Required.
The value to be tested.
Return Value
Type Description
A boolean. true if the value is Number.NaN, otherwise false.
    Notes:
  • Difference Between isNaN() and Number.isNaN()
    • isNaN() method returns true if a value is Not-a-Number.
    • Number.isNaN() returns true if a number is Not-a-Number.
    • In other words: isNaN() converts the value to a number before testing it.
  • Number.isNaN() is an ECMAScript6 (ES6) feature.
  • ES6 (JavaScript 2015) is supported in all modern browsers since June 2017.
  • Not supported in Internet Explorer.

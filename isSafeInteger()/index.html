JavaScript Number Method: .isSafeInteger()

    Definition:
  • this method returns true if a number is a safe integer.
  • Otherwise it returns false.

Syntax

Number.isSafeInteger(value)

Similar to:

  • Number.isInteger() Method
  • MAX SAFE INTEGER Property
  • MIN SAFE INTEGER Property
Parameters
Parameter Description
value Required.
The value to be tested.
Return Value
Type Description
A boolean true if the value is a safe integer, otherwise false.
    Notes:
  • Number.isSafeInteger() is an ECMAScript6 (ES6) feature.
  • ES6 (JavaScript 2015) is supported in all modern browsers since June 2017
  • Number.isSafeInteger() is not supported in Internet Explorer.
    Reference:
  • W3schools

Example

Code:

let result1 = "Is 123 a safe integer? " + Number.isSafeInteger(123) + "<br>" +
"Is -123 a safe integer? " + Number.isSafeInteger(-123) + "<br>" +
"Is '123' a safe integer? " + Number.isSafeInteger('123');

document.getElementById("example1").innerHTML = result1;

let result2 =
"Is 253 a safe integer? " + Number.isSafeInteger(Math.pow(2, 53)) + "<br>" +
"Is 253 - 1 a safe integer? " + Number.isSafeInteger(Math.pow(2, 53) - 1);

document.getElementById("example2").innerHTML = result2;

let result3 =
"Is true a safe integer? " + Number.isSafeInteger(true) + "<br>" +
"Is false a safe integer? " + Number.isSafeInteger(false) + "<br>" +
"Is Infinity a safe integer? " + Number.isSafeInteger(Infinity) + "<br>" +
"Is -Infinity a safe integer? " + Number.isSafeInteger(-Infinity);

document.getElementById("example3").innerHTML = result3;
